## Run Locally

Clone the project

```bash
  git clone https://github.com/CsProjectGroup554-Inferno/CS554_FinalProject
```

Go to the project directory

```bash
  cd CS554_FinalProject
```

Install dependencies

```bash
  npm install --force
```

Run Server

```bash
  npm start
```

Go to the client directory

```bash
  cd client
```

Install dependencies

```bash
  npm install --force
```

Run client

```bash
  npm start
```
